# Golf Performance Tracker

Scaffolded minimal Next.js + TypeScript project for the Golf Performance Tracker PWA.

Next steps:

- Run `npm install` to install dependencies.
- Create a `.env` with `DATABASE_URL` for Prisma.
- Run `npx prisma migrate dev --name init` to create the database schema.

Files added:

- `package.json`, `tsconfig.json`, `next.config.js`
- `pages/` basic pages and `_app.tsx`
- `styles/globals.css`, Tailwind/PostCSS config
- `public/manifest.json`, `public/sw.js`
- `prisma/schema.prisma`
